# python-port-scanner
My first cybersecurity project build usimg python.
This program scans ports on a target host and shows which ports are open. It uses python socket library to attempt connections to ports from 1 to 1024.
## Technologies used are- Python and Socket Library
## How to run- 1)Install python on your device. 2)Download the project from GitHub. 3)Run the program using: python port_sanner.py 4)Enter a target IP Address or host name when prompted.
## You can also test with:- 127.0.0.1 OR scanme.nmap.org
